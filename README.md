# TCS Website Clone 🖥️

A near pixel-perfect clone of [tcs.com](https://www.tcs.com) built entirely 
with pure HTML & CSS — no JavaScript, no frameworks, no libraries.

## 🔨 What's Built

- Fixed navigation header with TCS & TATA logos
- Full-screen background video hero section
- Horizontally scrolling "What's New" news cards
- "Cutting Edge Solutions" dark section with 3 service cards
- Customer Stories section with gradient image cards
- "Meet Us Here" events section
- News & Insights section with 4 spotlight image cards
- Careers section with image and CTA
- "With You for the Long Run" CTA section
- Footer with logo, legal links, and social icons

## 🎨 Techniques Used

- CSS `position: absolute` based layouts
- `linear-gradient` overlays on background images
- Custom `@font-face` fonts (DMSans, Neue Haas, Houschka)
- Fixed video background with `object-fit: cover`
- Font Awesome icons for social media links
- Hover transitions on buttons and links

## 🚀 Built With

- HTML5
- CSS3
